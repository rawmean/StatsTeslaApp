Need help with Stats?

The best place to start is the Support button inside the Stats app. It helps route your request with the right context and is the fastest path for most issues.

Best support path

Open the Stats app and use the in-app Support button first whenever possible. That path is preferred because it helps provide the right context for troubleshooting.

If you still need to reach out directly, email us and include a short description of the issue, your Tesla model, and any relevant screenshots or steps.
